Python Full Stack Foundation
Duration: 10 Weeks (2.5 Months)
Schedule: Monday–Friday  
Class Timings: 7:00 PM IST – 8:00 PM IST
Daily Session: 1 Hour 
Scope: Python Basics, Intermediate Python, OOPs, Useful Modules, SQL, HTML, CSS, Flask, Streamlit, APIs
Pace: Moderate
Level: Beginner to Intermediate

Week 1 – Python Fundamentals
Day 1: Introduction to Programming, Problem Solving, Python Installation, VS Code Setup, First Program, print()
Day 2: Variables, Data Types, Naming Rules, Comments, Input(), Output()
Day 3: Type Casting, Arithmetic Operators, Assignment Operators
Day 4: Relational Operators, Logical Operators, Membership Operators, Identity Operators
Day 5: Expressions, Operator Precedence, Mini Practice Problems, Calculator Project

Week 2 – Decision Making & Loops
Day 6: if Statement, Conditions, Indentation
Day 7: if-else Statement, Nested if
Day 8: if-elif-else Statement, Multiple Conditions
Day 9: for Loop, range(), Loop Patterns
Day 10: while Loop, break, continue, pass

Week 3 – Functions & Collections
Day 11: Functions, Function Definition, Function Calling
Day 12: Parameters, Arguments, Return Statement, Scope
Day 13: Lambda Functions, Anonymous Functions
Day 14: map(), filter(), reduce(), Practical Examples
Day 15: Lists, Indexing, Slicing, List Methods

Week 4 – Collections & Modules
Day 16: Tuples, Sets, Set Operations
Day 17: Dictionaries, Nested Dictionaries, Dictionary Methods
Day 18: math Module, sqrt(), pow(), factorial(), ceil(), floor(), pi
Day 19: random Module, randint(), choice(), shuffle()
Day 20: datetime Module, Date Operations, Time Operations

Week 5 – Files, OS & Regular Expressions
Day 21: Strings, String Methods, String Formatting
Day 22: File Handling, Read, Write, Append
Day 23: Exception Handling, try, except, finally
Day 24: os Module, Folder Operations, File Operations
Day 25: re Module, Pattern Matching, Email Validation, Mobile Validation

Week 6 – APIs & OOP Fundamentals
Day 26: json Module, JSON Parsing, JSON Read/Write
Day 27: requests Module, APIs, GET Requests, JSON Response
Day 28: Classes, Objects, Attributes
Day 29: Constructors, init(), Instance Variables
Day 30: Methods, Self Keyword, Object Interaction

Week 7 – OOP Intermediate
Day 31: Class Variables, Static Methods
Day 32: Inheritance
Day 33: Method Overriding, Multiple Inheritance
Day 34: Polymorphism
Day 35: Encapsulation, Abstraction

Week 8 – SQL Fundamentals
Day 36: Introduction to Databases, DBMS, RDBMS
Day 37: CREATE DATABASE, CREATE TABLE, Data Types
Day 38: INSERT, UPDATE, DELETE
Day 39: SELECT, WHERE, ORDER BY
Day 40: Aggregate Functions, GROUP BY, HAVING

Week 9 – SQL Intermediate & Front End
Day 41: Joins, Inner Join, Left Join, Right Join
Day 42: Subqueries, Constraints, Views
Day 43: HTML Structure, Headings, Paragraphs, Lists, Links
Day 44: Tables, Forms, Images, Input Controls
Day 45: CSS Basics, Selectors, Colors, Fonts, Box Model

Week 10 – Flask & Streamlit
Day 46: Flask Installation, First Flask Application, Routing
Day 47: Templates, Jinja2, HTML Integration
Day 48: Forms, User Input, Flask + SQL Integration
Day 49: Streamlit Installation, Widgets, Data Display, Charts
Day 50: Final Project Architecture, GitHub Upload, Deployment Basics
